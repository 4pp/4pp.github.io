老话说

发表于 2017-01-05 | 分类于 生活

生活需要总结，在此备忘，已便认清和警醒自己。

做人

好好学习，天天向上。

与其彷徨，不如每一天干点实事，活得扎实。

三十做到而立、四十做到不惑。

做事

工欲善其事必先利其器。

机会 = 积累 + 准备 + 小小的运气

先制定并实现一个小目标。

质变都来自于量变的积累。

学习靠积累，记忆靠理解，经验靠反思，熟练靠练习。

学习

泰山不拒细壤，故能成其高；江海不择细流，故能就其深。

不积跬步，无以至千里；不积小流，无以成江海
